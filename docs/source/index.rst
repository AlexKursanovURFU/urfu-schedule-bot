Добро пожаловать в документацию urfu-schedule-bot!
====================================================

Бот для расписания Уральского федерального университета (УрФУ)

Команды бота
------------

* /start - Приветствие и список команд
* /help - Подробная справка
* /about - Информация о боте
* /schedule - Показать расписание
* /setgroup - Выбрать группу
* /mygroup - Показать текущую группу
* /cleargroup - Очистить группу
* /days - Установить количество дней
* /groups - Список предустановленных групп
